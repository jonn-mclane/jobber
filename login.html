Página de Login

Pagina para acesso dos usuários cadastrados, funcionará tanto para os candidatos, quanto para as empresas.
Conta com um input para o username, e outro para sua senha, mais um botão de login para os usuários que já fazem parte do sistema.
Caso seja um novo usuário que ainda não se cadastrou, a página conta também com um anchor de cadastro, detinado a mesma página que será utilizada para edição do perfil do usuário.
Além disso, possui um botão para recuperação de senha.

Clique na parte cinza, ou no ícone "i" azul à direita para uma pré visualização de como será a página.

Clique no botão home no topo para voltar ao mapa do site.

O botão de login está temporariamente direcionado à futura página principal do aplicativo. Usuário e senha temporariamente desabilitados.
